# Goalkeeper Performance Analysis

## Overview

This project investigates which goalkeeper statistics are most strongly associated with Clean Sheets per 90 using Microsoft Excel.

The analysis demonstrates practical data analysis techniques including cleaning, correlation analysis, dashboards and professional reporting.

---

## Skills Demonstrated

- Microsoft Excel
- Data Cleaning
- Correlation Analysis
- Pivot Tables
- Conditional Formatting
- Dashboard Design
- Data Visualisation
- Report Writing

---

## Key Findings

- Goals Prevented had one of the strongest positive relationships with Clean Sheets per 90.
- Save Percentage also showed a strong association.
- Crosses Stopped demonstrated only a weak relationship.
- Correlation indicates association rather than causation.
